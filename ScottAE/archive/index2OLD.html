Scott Stompbox
This project started out three years ago as a simple attempt to change a guitar effects processor's patch via a foot controlled stompbox. My first attempt was little more than a switch debounce circuit sending a midi command from an 8 bit mcu. After a number of design enhancements and build iterations, I am confident that this stompbox compete's well with similar commercially available stompbox. One sad fact I learned from building these stompbox prototypes is; as an individual, the raw materials necessary to build this stompbox are more expensive than a fully assembled unit from some of the lower cost providers. This fact alone will liekly prevent me from mass producing this stompbox, but it does provide useful information for other DIY guitar accessory enthusiasts.

Scott Stompbox Features:
1:  Pedalboad provides 16 independently assignable foot switches used to send MIDI program change, control change, .... sys ex messages.
      (see http://en.wikipedia.org/wiki/MIDI)
2:  Windows based control program (download installer below) lets you assign MIDI functions to each footswitch
3:  Easily toggle between 4 configurations in realtime providing (4x16) 64 available foot commands.
3:  Midi transmit and receive function via standard midi cables or USB cable. (midi yoke)
4:  Stompbox is compatible with most DAWs.
5:  Buttons can be configured as latch (toggle on/off) or pulse function
6:  Pedalboard has an Atmel 8-bit AVR RISC-based microcontroller (ATmega168) dedicated to monitoring footswitch press.
      (see http://arduino.cc/en/Main/ArduinoBoardProMini)
7:  Stompbox has an Atmel SAM3X8E ARM Cortex-M3 CPU microprocessor dedicated to footswitch processing, midi processing, display processing, and SD file processing.
      (see http://arduino.cc/en/Main/arduinoBoardDue)
8:  Stompbox has an AdaFruit Micro SD memory card reader equipeed with a 2 gig Micro SD memory card to retain configuration information while poewered off.
      (see http://www.adafruit.com/product/254)
9:  Stompbox has a large 4 line, 20 character, blue backlit display is easy to read at a distance.
      (see http://www.hantronix.com/files/data/127855682920416lb.pdf)
10:  Custom Reaper 'Action List' is provided to demonstarte the capabilities of the stompbox with Reaper(download provided below)

Reaper 'Action Lists' commands:
1: Transport commands: start, stop, pause, rewind, fast foward, record arm. start/stop recording.
2: Marker commands: set markers in your Repaer project and jump to a marker via a switch press.
3: Looping commands: press a foot switch set loop begin points and loop endpoints.
4: Speed commands: press a foot switch to increase or decrease the tempo of a loop.
5: Track commands: press a foot switch to scro/l through tracks and arm/disarm tracks for recording .
