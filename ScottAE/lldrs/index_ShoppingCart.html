As an individual, I have been developing retail, financial, distribution, and manufacturing software for Fortune 500 companies since 1976. Since 1998, my primary focus has been web application development with a more recent focus on web retail store-fronts. Our products include software to manage and publish web product catalogs, web shopping carts, a First Data credit card payment processing gateway, a UPS Shipping rate calculation gateway, and extensive back office product management and order processing software. Security, stability, scalability, and ease of use are the cornerstone of our system designs.

Our web retail store-front software was designed to run on your servers and was developed using 100% 'open source' software (ie: Java programming language, Sun's Glassfish Sun's Glassfish Java Application Server, MySQL database management system, and RedHat Enterprise Linux operating system). Our software is provided 100% free of charge to individuals or corporations and includes program source code and database schema. An installation guide is provided to guide you through the implementation process and an online tutorial provides basic system usage instruction.

The following steps outline the process to implement a LLDRS retail store-front.

 1:  A PC server (Pentium 4, 4Gb memory, 500Gb disk) is procured.
 2:  A Red Hat Enterprise Linux (V7) operating system is installed.
 3:  A DSL connection to the internet is configured.
 4:  A MySQL database server software is installed.
 5:  A Verisign or GoDaddy security certificate is installed.
 6:  LLDRS web shopping cart software is installed.
 7:  LLDRS credit card payment processing gateway is configured.
 8:  Your products are setup and published in your product catalog.
 9:  Your new web store-front is linked to your existing website.
10:  Your products are now available for purchase over the web!
11:  You manage and ship your product orders.
12:  Your merchant account is debited with your product sales!


Items 3,5, and 7 have recurring monthly charges estimated to total approximately $100/month.

LLDRS web retail store-front can be configured to work with any credit card processing gateway (requires programming). A free First Data credit card processing gateway module is available after signing a LLDRS non-disclosure agreement. First Data is the largest credit card processor in the United States. First Data charges are approx $100 to set up and $15/month plus 'per transaction' fees

LLDRS web retail store-front also includes an optional UPS shipment rating module to calculate shipping charges during the shopping cart checkout process. If the customer selects a UPS shipping option during the checkout process, UPS shipping charges are calculated (via a UPS web service) and added to the customer order. A UPS tracking number is attached to the order and provided on the customer order confirmation (email).
